desc: A guide on how to install TeaClient
---


Before you download and install, make sure you have the following specifications:

- Windows: 10 & later
- MacOS: Sierra & later
- Linux: Distrobuted as an AppImage

<h2 class="text-2xl font-bold pt-3">Downloads</h2>

<h4 class="text-md font-bold pt-2">Windows</h4>

For Windows, click <a class="text-blue-500" href="/download#windows">here</a> to download.

<h4 class="text-md font-bold">MacOS</h4>

For MacOS, click <a class="text-blue-500" href="/download#macos">here</a> to download.

<h5 class="text-sm font-bold pt-5">Package Managers</h5>

if you use Homebrew install TeaClient with the following command:
```sh
brew install teaclient
```
<br/>
if you use MacPorts install TeaClient with the following command:
```sh
sudo port install teaclient
```

<h5 class="text-sm font-bold pt-5">Nix</h5>
if you use Nix install TeaClient with the package below: 
```nix  
pkgs.teaclient
```
